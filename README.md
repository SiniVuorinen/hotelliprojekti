Hotellivarausjärjestelmä (C++)

Tämä on ohjelmoinnin perusteiden kurssin lopputyö, jossa toteutin hotellin varausjärjestelmän C++:lla.
Ohjelma toimii valikkopohjaisesti ja mahdollistaa huoneiden varaamisen, varausten perumisen, hakemisen ja muokkaamisen.
Huoneiden tila tallennetaan huoneet.txt-tiedostoon, jotta varaukset säilyvät myös ohjelman sulkemisen jälkeen.
